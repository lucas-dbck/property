# YieldDesk Chrome Extension

This extension imports the Immoweb listing currently open in Chrome into YieldDesk.

## Install locally

1. Open Chrome.
2. Go to `chrome://extensions`.
3. Turn on `Developer mode`.
4. Click `Load unpacked`.
5. Select this folder:
   `chrome-extension`

## Use

1. Open an Immoweb listing in Chrome.
2. Click the YieldDesk extension button.
3. Check the YieldDesk app URL.
4. Click `Import current listing`.

The extension opens YieldDesk on the Analyze page with the fields prefilled.
